#### YouTube Video Transcripts (`youtube-transcripts/`)
**Purpose:** Collect and store transcripts from latest 3 relevant videos per expert  
**Format:** Markdown with time-coded sections  
**Per Expert Contains:**
- 3 most recent or most relevant videos
- Video titles and URLs
- Publication dates and duration
- Full transcripts (or key sections)

**Status:** 1/10 started (Neil Patel with 1 complete transcript + 2 placeholders)  
**Template:** [TEMPLATE.md](youtube-transcripts/TEMPLATE.md)

**How to Collect YouTube Transcripts:**

**Option A: Using YouTube Auto-Generated Captions**
1. Go to the YouTube video
2. Click the three dots menu → "Show transcript"
3. Copy the entire transcript
4. Format and paste into the markdown file

**Option B: Using Third-Party Services**
- **Rev.com:** Professional transcription (paid, ~$1.25/minute)
- **YouTube Transcript API:** Automated extraction (free if using Python)
- **FrameIO:** Collaborative video review and transcription

**Option C: AI Tools**
- Use ChatGPT or Claude to summarize/transcribe from YouTube video summaries

**Steps:**
1. Visit each expert's YouTube channel (link in sources.md)
2. Identify 3 most recent videos related to AI-powered SEO or B2B content
3. Extract transcripts using preferred method
4. Format with timestamps and section headers
5. Follow the template in [TEMPLATE.md](youtube-transcripts/TEMPLATE.md)
